# Sistema Bancário em Python

Projeto desenvolvido durante a formação Python da DIO, com o objetivo de praticar lógica de programação, estruturas de repetição, condicionais e manipulação de dados em Python.

## Funcionalidades

* Depósito
* Saque
* Extrato bancário
* Controle de limite de saque
* Controle de quantidade de saques diários

## Tecnologias utilizadas

* Python

## Objetivo do projeto

Aplicar conceitos fundamentais de Python através da construção de um sistema bancário simples via terminal.

## strutura do Projeto

* `desafio_v1.py` → versão inicial do sistema bancário com depósito, saque e extrato.
* `desafio_v2.py` → versão modularizada com funções, cadastro de usuários e contas bancárias.
